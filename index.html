Estado: Local (este navegador)

Para que todo el equipo edite el mismo roadmap en tiempo real, conéctalo a Firebase (gratis, sin tarjeta). Sigue estos pasos una sola vez:

  1. Crea una cuenta Google nueva dedicada y entra a console.firebase.google.com.
  2. «Agregar proyecto» → nómbralo (ej. roadmap). Salta Google Analytics.
  3. Menú Build → Firestore Database → Crear base de datos → modo producción → ubicación más cercana.
  4. Pestaña Reglas de Firestore: pega esto y publica (deja la edición abierta, sin caducidad): 
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
  5. Engranaje ⚙️ → Configuración del proyecto → «Tus apps» → icono web </> → registra la app → copia el objeto firebaseConfig y pégalo abajo.

Después, para publicarlo gratis y para siempre: sube este archivo como index.html a un repositorio público de GitHub y activa Settings → Pages. Pega tu config en el archivo (botón «Copiar para pegar») antes de subir, así funciona para todos.
